# Roadmap propuesto

## Fase 1. Descubrimiento

- Alinear alcance de negocio
- Priorizar journeys
- Definir stakeholders
- Establecer KPIs de exito

## Fase 2. POC funcional

- Configurar journeys demo
- Crear datos y expedientes mock
- Simular integraciones clave
- Montar dashboards base

## Fase 3. Validacion

- Ejecutar sesiones con usuarios clave
- Ajustar reglas y mensajes
- Medir comprension y valor percibido
- Documentar learnings

## Fase 4. Piloto

- Conectar integraciones reales
- Definir modelo operativo
- Endurecer seguridad y gobierno
- Preparar despliegue controlado
